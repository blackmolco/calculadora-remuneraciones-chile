# Guía de Instalación para el Calculador de Nómina en Excel

## Requisitos Previos
- Tener Microsoft Excel instalado en su computadora.
- Acceso a Internet para descargar archivos.

## Paso 1: Descargar el archivo
1. Visite el repositorio de GitHub: [calculadora-remuneraciones-chile](https://github.com/blackmolco/calculadora-remuneraciones-chile).
2. Haga clic en el botón **Code** y seleccione **Download ZIP**.
3. Descomprima el archivo ZIP en su computadora.

## Paso 2: Abrir el archivo en Excel
1. Navegue a la carpeta descomprimida.
2. Haga doble clic en el archivo `calculadora_nomina.xlsx` para abrirlo en Microsoft Excel.

## Paso 3: Configurar el archivo
1. Revise las instrucciones en la hoja de cálculo para entender cómo se estructuran los datos.
2. Ingrese los datos de sus empleados en las secciones correspondientes.
3. Asegúrese de que todos los datos estén completos y correctamente formateados.

## Paso 4: Calcular la nómina
1. Una vez que haya ingresado toda la información, dirígete a la hoja de **Cálculo**.
2. Haga clic en el botón **Calcular** (si está disponible) o revise las fórmulas para asegurarse de que calculen correctamente los salarios.

## Paso 5: Guardar los resultados
1. Después de verificar los cálculos, guarde el archivo para conservar los datos.
2. Use **Guardar como** para crear una copia si desea preservar el archivo original.

## Paso 6: Consultar documentación adicional
- Para más detalles, consulte el archivo README.md en el repositorio o la sección de ayuda en Excel.

## Soporte
Si encuentra problemas, abra un problema en el repositorio de GitHub o contacte a la comunidad para obtener asistencia.
